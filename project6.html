Mobile Price Analysis And Classification





Introduction:


In this project, I conducted an in-depth analysis of a mobile phone dataset with the goal of understanding the factors influencing the price range of mobile devices. The analysis involves exploratory data analysis (EDA), data visualization, feature selection, and the application of various classification algorithms to predict the price range of mobile phones.



Data Exploration:


Sample Data:

Loaded the dataset using Pandas.

I began by loading and exploring the dataset using the pandas library. Key steps included checking for duplicates, handling missing values.






Key Visualizations:


visualizing relationships between variables. Notable visualizations include line plots, count plots, box plots, and heatmaps to uncover insights into features such as RAM, camera specifications, and connectivity options.






Feature Selection:


Importance of Feature Selection:

To identify the most relevant features for predicting price range, I applied SelectKBest, Extra Trees Classifier, and Information Gain techniques. This process involved evaluating the significance of each feature and selecting the top ones. The selected features were then visualized to highlight their importance.






Selected Features:

The selected features, visualized through bar plots and heatmaps, played a crucial role in improving the model's accuracy.






Classification Models:


Following feature selection, I implemented several classification algorithms, including Logistic Regression, Support Vector Machine (SVM), K-Nearest Neighbors (KNN), Naive Bayes, and Decision Tree Classifier. I assessed the performance of each model using accuracy scores and confusion matrices.

Model Evaluation:


Accuracy Scores:

I evaluated each model's performance and calculated accuracy scores. This allowed for a comprehensive comparison of the models.






Model Comparison:

To provide a comprehensive overview of model performance, I created a bar plot comparing the accuracy of different classification models. This visualization helps in understanding the relative strengths of each algorithm in predicting mobile price ranges.






Conclusion:

The project demonstrates the process of analyzing and classifying mobile phone prices based on various features. The combination of exploratory data analysis, feature selection, and classification models provides valuable insights for both understanding the dataset and building predictive models for mobile price range classification.

Data Cleaning and Analysis in Cricket Statistics

Project Description

In this project, I tackled the intricacies of cleaning and analyzing a dataset containing cricket player statistics sourced from ESPNcricinfo.

The primary goal was to enhance the dataset's quality for more accurate insights into cricket players' performance.

Code Organization

The code is organized using Python and pandas for efficient data manipulation, with detailed comments explaining each step.

Additional Details

The dataset presented challenges like missing values and inconsistent formats, which I methodically addressed.

Displaying Results

The analysis focused on cricket player statistics, using advanced statistical calculations to derive meaningful insights.

  • Q1: Calculated the average career length of players.
  • Q2: Determined the average batting strike rate for cricketers with over 10 years of play.
  • Q3: Found the number of cricketers who played before 1960.
  • Q4: Identified the maximum highest innings score by country.
  • Q5: Computed the average points scored by players for each country.

Final Results

After meticulous analysis, the focus was on achieving clean and error-free data for reliable insights into cricket player performances.
